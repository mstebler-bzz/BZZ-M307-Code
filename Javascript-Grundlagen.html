JavaScript Grundlagen

1. Datentypen

JavaScript hat verschiedene Datentypen:

// String
let name = "Max";
console.log(name); // Ausgabe: Max

// Number
let zahl = 42;
console.log(zahl); // Ausgabe: 42

// Boolean
let istWahr = true;
console.log(istWahr); // Ausgabe: true

2. Funktionen und Parameter

Funktionen helfen, Code wiederzuverwenden:

function begruessung(name) {
    return "Hallo, " + name + "!";
}
console.log(begruessung("Anna")); // Ausgabe: Hallo, Anna!

3. Objekte

Objekte speichern Daten in Schlüssel-Wert-Paaren:

let person = {
    vorname: "Lukas",
    alter: 25
};
console.log(person.vorname, person.alter); // Ausgabe: Lukas 25

4. Arrays

Arrays speichern Listen von Werten:

let farben = ["Rot", "Grün", "Blau"];
console.log(farben[0]); // Ausgabe: Rot

5. For-Schleifen

Schleifen helfen, Code mehrfach auszuführen:

for (let i = 0; i < farben.length; i++) {
    console.log(farben[i]);
}
// Ausgabe:
// Rot
// Grün
// Blau

6. DOM-Manipulation

Ändern und Interagieren mit der Webseite:

Element mit ID abrufen

// HTML
<h2 id="title">Alter Titel</h2>

// JavaScript
let ueberschrift = document.getElementById("title");
console.log(ueberschrift.textContent); // Ausgabe: Alter Titel

Textinhalt setzen

// HTML
<p id="text">Alter Text</p>

// JavaScript
let textElement = document.getElementById("text");
textElement.textContent = "Neuer Text";
console.log(textElement.textContent); // Ausgabe: Neuer Text

Mit querySelector

// HTML
<div class="container">Originaler Inhalt</div>

// JavaScript
let container = document.querySelector(".container");
console.log(container.textContent); // Ausgabe: Originaler Inhalt

Click-Event hinzufügen

// HTML
<button id="alertButton">Klick mich</button>

// JavaScript
let alertButton = document.getElementById("alertButton");
alertButton.addEventListener("click", function() {
    alert("Button wurde geklickt!");
});
